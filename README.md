# MYP Grade 7 Revision Quizzes

This project provides separate revision quizzes for:
- Sciences
- Mathematics
- Individuals and Societies

Each quiz includes:
- Grade 7 level multiple-choice questions
- Immediate answer reveal
- Short explanation to support understanding
- Subject score summary
- Mobile-friendly layout for phones and tablets

## Local Run

Open `index.html` in any modern browser.

## Host Publicly with GitHub Pages

1. Create a new GitHub repository (public).
2. Push this project to the repository.
3. In GitHub, open repository Settings -> Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
5. Save, then wait for deployment.
6. Your public link will look like:
   - `https://YOUR_USERNAME.github.io/YOUR_REPOSITORY_NAME/`

## Suggested Repository Name

- `myp-grade7-revision-quizzes`
